## 🤖 Bot Intelligence

### Keyword Matching
The bot uses sophisticated regex patterns to match user intent:
- Multiple keyword variations
- Context-aware responses
- Fuzzy matching for typos
- Multi-language support (English, Hindi terms)

### Response Randomization
- Each category has multiple response variations
- Random selection prevents repetitive answers
- Natural conversation flow
- Varied response styles

### Conversation Flow
1. User sends message
2. Bot analyzes keywords and context
3. Matches to appropriate category
4. Selects random response from category
5. Simulates typing delay (1-2 seconds)
6. Delivers personalized response
